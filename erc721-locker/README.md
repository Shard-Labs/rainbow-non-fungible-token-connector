# ERC721 Locker

The entry point for locking up tokens that get migrated to Near

![](https://i.imgur.com/DavgJDz.png)

### Installing Dependencies

```
yarn
```

### Running tests

```
yarn test
```

### Running coverage

```
yarn coverage
```
